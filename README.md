# 📊 RaloZero: Financial Intelligence Dashboard

O **RaloZero** é um dashboard de BI desenvolvido para transformar planilhas financeiras em decisões estratégicas de alto impacto.

## 🚀 Funcionalidades
- **Métricas em Tempo Real:** Visualização imediata de Faturamento, Despesas e Lucro Líquido.
- **Análise de Gastos:** Gráficos interativos que identificam os maiores "ralos" financeiros por categoria.
- **Filtros Inteligentes:** Seleção por período e categorias específicas.
- **Exportação:** Botão para download de relatórios filtrados em CSV.

## 🛠️ Tecnologias
- Python 3.11+
- Streamlit (Interface Web)
- Pandas (Processamento de Dados)
- Plotly (Gráficos Interativos)

## 📦 Como rodar o projeto
1. Instale as dependências: `pip install -r requirements.txt`
2. Execute o app: `python -m streamlit run app.py`

3. Obrigatorio
streamlit
pandas
plotly
